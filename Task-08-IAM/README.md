# Task 8: Create IAM Users and Groups

## Objective
To understand IAM user and group-based access control.

## Services Used
- AWS IAM

## Steps Performed
1. Created IAM group named "Developers".
2. Attached AmazonS3ReadOnlyAccess policy to the group.
3. Created IAM user "dev-user".
4. Added user to Developers group.

## Verification
- User successfully created.
- Group shows attached S3 policy.
- User belongs to Developers group.

## Outcome
Successfully implemented group-based permission management using IAM.
