# Task 23: Azure Resource Manager (ARM) & Cost Management

## Objective
Understand Resource Groups, ARM structure, and analyze subscription-level costs.

## Services Used
- Azure Resource Manager (ARM)
- Resource Groups
- Azure Subscription
- Cost Management & Cost Analysis

## Steps Performed

1. Opened Resource Groups and selected **ppt-rg**.
2. Viewed all resources inside the Resource Group:
   - Virtual Machine
   - Virtual Network
   - Disk
   - Public IP
   - NSG
   - Storage Account
3. Verified ARM structure organization.
4. Opened **Subscriptions → Azure subscription 1**.
5. Navigated to **Cost Analysis** under Cost Management.
6. Reviewed cost analysis dashboard at subscription scope.

## Observations
- ARM organizes related resources inside a Resource Group.
- Cost Analysis provides subscription-level cost monitoring.
- In sandbox/free tier, cost data may show limited or unavailable usage.

## Outcome
Successfully explored ARM structure and Azure Cost Management.
