# Task 19 – Azure Storage Account

## Objective
To create an Azure Storage Account and explore Blob, File, and Table storage services.

## Steps Performed
1. Created a Storage Account **pptstorage1901** in Resource Group **ppt-rg** (West US 2).
2. Selected **Standard performance** and **Locally-redundant storage (LRS)**.
3. Created a Blob container **samplecontainer** and uploaded a sample file.
4. Created a File Share **samplefileshare**.
5. Created a Table **sampletable**.

## Output
- Storage Account deployed successfully.
- Blob container created and file uploaded.
- File share created successfully.
- Table storage created successfully.

## Conclusion
Azure Storage Account was successfully configured and Blob, File, and Table storage services were implemented.
