# Task 6: S3 Replication and Lifecycle Policy

## Objective
To configure S3 cross-region replication and lifecycle policies for backup and cost optimization.

## Steps Performed
- Created source and destination buckets in different regions.
- Enabled versioning on both buckets.
- Configured replication rule.
- Verified replication by uploading a test file.
- Created lifecycle rule to transition objects to Standard-IA.

## Outcome
Successfully implemented S3 cross-region replication and lifecycle policy for storage optimization and backup strategy.
