# Task 1 – Static Web Hosting using EC2 (Ubuntu + Nginx)

## Objective
To deploy a static website on an Ubuntu EC2 instance using Nginx web server.

## Steps Performed
1. Launched Ubuntu EC2 instance.
2. Configured Security Group (SSH & HTTP).
3. Installed Nginx web server.
4. Downloaded and deployed website template.
5. Accessed website using public IP address.

## Result
Website successfully hosted and accessible via browser.
