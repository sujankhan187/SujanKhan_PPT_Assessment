# Task 2: Disk Partitioning (Ubuntu – MBR & Windows – GPT)

## Objective
To understand disk partitioning using MBR and GPT formats.

## Steps Performed

### Ubuntu (MBR)
- Used fdisk to create disk partition.
- Verified partition using lsblk.
- Observed MBR partition table format.

### Windows (GPT)
- Used Disk Management tool.
- Created GPT partition.
- Verified partition structure.

## Outcome
Successfully created and verified disk partitions in both Ubuntu (MBR) and Windows (GPT).
