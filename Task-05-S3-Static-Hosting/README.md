# Task 5: Static Website Hosting in S3

## Objective
To host a static website using Amazon S3.

## Steps Performed
- Created an S3 bucket.
- Enabled static website hosting.
- Uploaded index.html file.
- Configured bucket policy for public access.
- Accessed website using S3 endpoint URL.

## Outcome
Successfully deployed a static website using Amazon S3 without using EC2.
