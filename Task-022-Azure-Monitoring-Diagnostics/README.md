# Task 22: Azure Monitoring & Diagnostics

## Objective
Monitor Azure resources using Azure Monitor, Metrics, Activity Log, and Alerts.

## Services Used
- Azure Monitor
- Metrics
- Activity Log
- Alerts

## Steps Performed

1. Opened Azure Monitor from Azure Portal.
2. Viewed Monitor Overview dashboard.
3. Opened Metrics section.
4. Selected VM (ppt-vm) as resource.
5. Viewed CPU usage graph.
6. Checked Activity Log for recent resource operations.
7. Explored Alerts section and verified alert rules.

## Observations
- Successfully monitored VM CPU usage.
- Activity log shows resource deployment and operations.
- No alert rules configured (default state).

## Outcome
Successfully explored Azure Monitoring and Diagnostics features.
