# Task 3: Attach EBS Volume from Different AZ using Snapshot

## Objective
To understand EBS snapshot creation and cross-AZ volume restoration.

## Steps Performed
- Created an EBS volume and attached it to EC2.
- Created snapshot of the volume.
- Restored snapshot as a new volume in a different Availability Zone.
- Attached new volume to EC2 and verified using lsblk.

## Outcome
Successfully created and restored EBS snapshot across Availability Zones.
