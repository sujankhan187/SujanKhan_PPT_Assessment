# Task 21 – Microsoft Entra ID (Users & Roles)

## Objective
To create a new user in Microsoft Entra ID and assign a basic administrative role for identity management practice.

## Services Used
- Microsoft Entra ID
- Users
- Roles and Administrators

## Steps Performed
1. Logged in to Azure Portal.
2. Opened Microsoft Entra ID.
3. Navigated to Users section.
4. Clicked New User → Create new user.
5. Entered user details:
   - User Principal Name: pptuser1
   - Display Name: PPT User
6. Clicked Review + Create and created the user successfully.
7. Navigated to Roles and Administrators.
8. Selected Global Reader role.
9. Clicked Add assignments and selected pptuser1.
10. Verified role assignment under Global Reader → Assignments.

## Output
- New user pptuser1 created successfully.
- Global Reader role assigned successfully.
- Role assignment verified in Azure Portal.

## Conclusion
The Microsoft Entra ID user was successfully created and assigned a Global Reader role. Identity and role management were verified successfully.
